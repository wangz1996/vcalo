Introduction¶

vcalo is a simulation and analysis framework designed for high-energy physics experiments. It provides tools for electromagnetic calorimeter simulation and data analysis.

Features¶

  • Support for complex detector simulations.

  • Flexible configuration using YAML files.

  • Integrated analysis tools.
