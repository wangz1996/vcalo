## Environment
Use cvmfs
```
	source /cvmfs/sft.cern.ch/lcg/views/LCG_105/x86_64-centos7-gcc12-opt/setup.sh
```

## Usage
Run simulation
```
	vcalo -c default.yaml
```

Print default config yaml
```
	vcalo -p
```

Print help message
```
	vcalo -h
```
